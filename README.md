# Titanic EDA

Exploratory data analysis on the Kaggle Titanic dataset.

## What I did
- Cleaned missing values (Age filled with median, Cabin dropped)
- Visualized survival rates by gender, class, and age group

## Key findings
- Women survived at ~74% vs men at ~19%
- First class passengers had significantly higher survival rates
- Children had the highest survival rate across all age groups

## Tools
Python, Pandas, Matplotlib
